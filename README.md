## ✨ Overview

The **Modern Intranet Kit** is a ready-to-deploy SPFx solution that delivers multiple powerful web parts through a **single `.sppkg` deployment**.

Designed for real-world enterprise use, it enables teams to create engaging, branded intranet experiences—without relying on external tools or services.

---

## 🔑 Key Highlights

- 🧩 **Modular Web Parts** — Build flexible, modern layouts  
- 📦 **Single Deployment** — Install everything at once  
- 🎨 **Native Theming** — Automatically adapts to SharePoint themes  
- 🔗 **Cross-Site Data** — Aggregate content across your tenant  
- 📌 **Smart Content Prioritisation** — Highlight important updates with built-in logic  

---

## 🛠️ Tech Stack

- **SPFx** 1.22  
- **React** 17 + TypeScript  
- **Fluent UI**  
- **Microsoft Graph + SharePoint REST API**  
- **Node.js + Heft + Webpack**  

---

## 💡 Why It Matters

This kit helps standardise intranet experiences while giving content authors **full control over layout, content, and branding**—without adding complexity.
